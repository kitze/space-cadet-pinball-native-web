# Notice

This project is an experimental native web prototype inspired by and derived from the open-source reverse-engineered `SpaceCadetPinball` project by Andrey Muzychenko:

https://github.com/k4zmu2a/SpaceCadetPinball

The original project is MIT licensed. This repository keeps the MIT license text.

Generated bitmap assets in `assets/` were created for this prototype with image generation prompts documented in `asset-prompts.md`. The original Microsoft game resource files are not included.
